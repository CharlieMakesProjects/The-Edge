The agent tech stack

Python scraping layer
BeautifulSoup + Playwright for SEC EDGAR, Capitol Trades, and Unusual Whales. Runs on a cron schedule (daily at 6am ET). Outputs structured JSON.
Claude API — content generation
Anthropic Claude (claude-sonnet-4-6) receives scraped data as context, generates briefing copy, sector analysis, and stock thesis updates. Structured outputs via XML tags.
beehiiv API — distribution
Agent pushes finished newsletter content to beehiiv via their REST API. Schedules and sends to your full subscriber list automatically. 0% revenue cut on paid subscriptions.
Vercel + GitHub — hosting
The static Hub (HTML files) lives on Vercel for free. GitHub Actions triggers the Python scraping agent on a daily cron job. Zero server cost until significant scale.
Supabase — data persistence
Free Postgres database stores scraped data, newsletter drafts, subscriber preferences, and agent run logs. Powers the live data feeds in The Hub once connected.
X API — social distribution
Agent auto-posts daily one-liner insights (Fear & Greed reading, top insider buy, sector rotation move) to X and LinkedIn, driving traffic back to The Hub and newsletter.
Bottom line: The goal is for The Edge to be a media company that largely runs itself — the agent scrapes the data, writes the content, sends the newsletter, and posts to social.
